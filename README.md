#☕ Coffee-Shop-Sales

## 🎯 Project Overview

This project is an interactive Coffee Shop Sales Dashboard created in Microsoft Excel. It helps analyze sales performance using Pivot Tables, Pivot Charts, and Slicers. The dashboard provides useful insights about sales, products, locations, and customer orders.

## 🛠️ Project Objective

The main objective  of this project is to:
- Clean and prepare raw sales data.
- Analyze sales performance.
- Create an interactive dashboard.
- Help users understand business performance through visual reports.

- ---

## 📂 Datasets 

The dataset contains Coffee Shop Sales information, including:
- transaction Date
- Product Name
- Store location
- Quantity Sold
- Unit Price
- Total Sales
